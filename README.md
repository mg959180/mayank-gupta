# Mayank Gupta Dev Portfolio

Personal developer portfolio website for Mayank Gupta. This project showcases a modern single-page portfolio experience with a responsive layout, animated hero section, portfolio filtering, contact section, and SEO-friendly metadata.

## Preview

This portfolio is built as a lightweight static site using plain HTML, CSS, and JavaScript, making it easy to customize, deploy, and maintain.

## Features

- Responsive portfolio landing page
- Typing effect in the hero section
- Sticky navigation with mobile menu support
- Portfolio gallery filtering interaction
- SEO meta tags for search and social sharing
- Externalized stylesheet and JavaScript for cleaner structure
- Social profile links for quick contact and discovery

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Google Fonts
- Font Awesome

## Getting Started

1. Clone the repository:

```bash
git clone <your-repo-url>
```

2. Open the project folder:

```bash
cd portfolio
```

3. Run the site locally by opening `index.html` in your browser.

If you want a simple local server, you can use:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`.

## Customization

- Update portfolio content in `index.html`
- Edit styling in `styles.css`
- Adjust animations and interactions in `script.js`
- Replace placeholder text and demo images with your real content
- Add your deployed domain to SEO tags like `canonical` and `og:url` when available
